<h1 align="left">📍 yatra-automation </h1></br>

Automated UI test suite for Yatra using Selenium WebDriver and Java (Maven).  </br>
This project contains scripts to validate key functionalities such as search, booking flows, and other user scenarios for the Yatra travel platform.

🚀 Features

✅ Cross-browser compatible test automation    </br>
✅ Modular Page Object Model (POM) design      </br>
✅ Maven-managed project with dependencies      </br>
✅ Easy test execution through Maven commands    </br>
✅ Scalable for adding new test suites            </br>

# Tech Stack

Language: Java  <br>
Test Framework: TestNG  <br>
Automation Library: Selenium WebDriver  <br>
Build Tool: Maven  <br>
Browser: Chrome
